# StartingMember サポートページ

サッカーチームのスタメンをかんたんに組んで共有できるiOSアプリです。

## 主な機能

- チーム・選手・スタッフの登録管理
- フォーメーション設定とスタメン編集
- スタメンカードを画像としてシェア
- スタメン履歴の保存

## プライバシーについて

StartingMember はプライバシーを最優先に設計されています。

- すべてのデータはお使いの端末内にのみ保存されます
- 外部サーバーへの送信は一切行いません
- 広告・トラッキングSDKは使用していません

## カメラ・写真ライブラリの使用について

- **カメラ**：選手・スタッフの写真撮影に使用します
- **写真ライブラリ**：選手・スタッフの写真登録に使用します
